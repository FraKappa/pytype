# pytyping
A simple typing speed test right in your terminal made with Python and the curses module.

![pytype](https://user-images.githubusercontent.com/72104527/112695896-235ca200-8e85-11eb-9d9a-e6c735f97278.gif)

## Installation
You can install pytyping using pip:
```
pip install pytyping
```

To run the program:
```
python -m pytyping
```
